# Product Roadmap

## Phase 0 — Stabilize Process Modeling Core

Goal:
Make Module 1 usable, stable, and polished.

Scope:
- Fix layout session-scoped scroll
- Fix AI Input Brief UX
- Fix File Intake PDF draft flow
- Redesign AI Connection Center
- Redesign Template Hub
- Simplify QA/PTR actions
- Clean duplicated placeholders and bilingual UI noise

Success criteria:
- User can generate Draft PTR, QA, BPMN, Service Blueprint, and export artifacts in under 5 minutes.
- No page-level horizontal scroll.
- AI output remains draft/recommendation/review before apply.

## Phase 1 — MVP with Real AI

Goal:
Release individual-first MVP for PO, BA, SA, Product Manager, and Architect users.

Scope:
- Stable Process Modeling Core
- Real AI Input/File/Chat to Draft PTR
- Rule QA + AI-assisted recommendations
- Basic BRD or User Story generation
- AI Coding Pack export
- Banking starter templates
- Local-first + BYOK-ready AI Connection Center
- Export Center

Recommended real AI skills:
- input-brief-to-ptr
- brd-or-notes-to-user-stories
- user-stories-to-ai-coding-pack
- process-qa-recommendation
