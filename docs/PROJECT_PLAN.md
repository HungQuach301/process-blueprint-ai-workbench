# MVP Project Plan

## Goal

Build a focused workbench that helps users turn process notes into a clear process blueprint.

## MVP Scope

- Create a simple Next.js application shell.
- Add an input area for raw process notes.
- Generate a structured process blueprint view from the notes.
- Allow users to review and refine the generated blueprint.
- Keep persistence local or minimal until the core workflow is validated.

## Initial Milestones

1. Scaffold the Next.js app skeleton.
2. Define the core blueprint data model.
3. Build the note input and blueprint preview workflow.
4. Add basic validation and empty states.
5. Prepare for AI integration after the local workflow is usable.

## App Sections

- Workspace
- Input Brief
- Template Library
- Process Task Register
- QA Panel
- D01 BPMN Preview
- D02 Service Blueprint Preview
- Export Center

## Non-Goals For MVP

- Multi-user collaboration.
- Advanced permissions.
- Complex workflow automation.
- Production deployment configuration.
